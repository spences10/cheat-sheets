# Useful Yarn commands

This is just stuff that I have put down that I find I use a lot of the
time for my own reference.

## Update dependencies

```
yarn upgrade
```

## displays the location of the yarn bin folder.

```
yarn bin
```

## list installed packages.

```
yarn ls
```
