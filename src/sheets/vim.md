## Exit Vim

Hit the Esc key to enter "Normal mode". Then you can type `:` to enter
"Command-line mode". A colon `:` will appear at the bottom of the
screen.

To quit:

```bash
:q
# enter
```

To quit without saving:

```bash
:q!
# enter
```
