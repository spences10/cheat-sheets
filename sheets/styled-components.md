## conditionally use a style

```jsx
<StyledButton
	active={true}
	onClick={this.handleButton}
></StyledButton>
```

And in your styles something like this:

```jsx
const StyledButton = styled.button`
	align-items: center;
	line-height: 0.2;

	${({ active }) =>
		active &&
		`
    background: blue;
  `}
`;
```
